generateFile(layoutFile, contentFile, outputFile, cb)

Generate a new file. It will write a new file to `outputFile` and call the callback when complete. The contents of that file are based on the `layoutFile` and the `contentFile`.
Parameters:
Name Type Description
layoutFile String The full path to the layout file.
contentFile String The full path to the content file.
outputFile String The full path to the output file.
cb SiteGenerator~generateFileCallback The callback.
